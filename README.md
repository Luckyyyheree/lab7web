# Belajar PHP Dasar

Repository ini berisi *contoh lengkap dasar-dasar PHP* dalam file HTML mandiri (bisa langsung dijalankan di browser via XAMPP/WAMP/Laragon). Cocok untuk pemula yang ingin belajar variabel, operator, kondisi, perulangan, hingga pengolahan form dengan GET dan POST.

---

## Daftar File & Fungsi

| File | Deskripsi | Output Contoh |
|------|-----------|---------------|
| php_dasar.php.html | Hello World pertama | Hello World |
| php_variable.php.html | Penggunaan variabel | NIM: 312410094<br>Nama: Ibnu Nazhif Alamsyah |
| operator.php.html | Operator aritmatika (gaji bersih) | Gaji sebelum pajak = Rp. 1000000<br>Gaji dibawa pulang = Rp. 900000 |
| perulangan.php.html | for, while, do-while | Perulangan ke-1 sampai ke-10 |
| Kondisi.php.html | if dan switch untuk hari | Senin (jika hari ini Senin) |
| Kondisi_IF.php.html | Menentukan grade berdasarkan nilai | Nilai Anda B (untuk 75) |
| Kondisi_Switch.php.html | switch untuk nama hari | Rabu |
| Predefined_Variable.php.html | Form GET | Input nama=Ani → Selamat Datang Ani |
| form_input.php.html | Form POST sederhana | Input nama=Budi → Selamat Datang, Budi! |
| form_post.php.html | Form lengkap: nama, tgl lahir, pekerjaan + hitung umur | Usia: 27 tahun |

---

## Cara Menjalankan

1. *Install XAMPP* → [https://www.apachefriends.org](https://www.apachefriends.org)
2. Taruh semua file di folder:  
   C:\xampp\htdocs\belajar-php-dasar\
3. Buka *XAMPP Control Panel* → Start *Apache*
4. Buka browser → Akses:  
   http://localhost/belajar-php-dasar/nama_file.html

> Semua file *bisa langsung dibuka dan diedit* — tidak perlu konfigurasi tambahan.
